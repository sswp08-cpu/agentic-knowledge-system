# Machine Learning

Machine Learning (ML) is a subset of [[Artificial Intelligence]] that enables systems to learn and improve from experience without being explicitly programmed.

## Key Concepts

### Definition
ML algorithms build mathematical models based on sample data (training data) to make predictions or decisions without being explicitly programmed to perform the task.

### Learning Paradigms

#### Supervised Learning
Learning from labeled examples to predict outcomes for new data.
- Classification (categorical outputs)
- Regression (continuous outputs)

#### Unsupervised Learning
Finding patterns in unlabeled data.
- Clustering
- Dimensionality reduction
- Anomaly detection

#### Reinforcement Learning
Learning through interaction with an environment to maximize rewards.
- Agent-based learning
- Policy optimization
- Q-learning

## Common Algorithms

| Type | Algorithms |
|------|------------|
| Supervised | Linear Regression, Decision Trees, SVM, Random Forest |
| Unsupervised | K-Means, DBSCAN, PCA, Autoencoders |
| Reinforcement | Q-Learning, Policy Gradient, Actor-Critic |
